# 🦚 Krishna Janmashtami Website

Krishna Janmashtami Website is a festive and visually engaging web project built using HTML, CSS, and JavaScript.  
It celebrates the festival of Janmashtami with beautiful design, animations, devotional elements, and an interactive user experience.

---

## 🚀 Features
- 🎨 Attractive festive UI design  
- ✨ Smooth animations and effects  
- 🦚 Krishna-themed visuals and styling  
- 📱 Fully responsive layout  
- ⚡ Interactive elements using JavaScript  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ How It Works
- Displays a Krishna Janmashtami themed interface  
- Uses animations and effects for engaging visuals  
- Responsive design works on both desktop and mobile devices
